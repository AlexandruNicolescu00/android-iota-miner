# android-iota-miner
android-iota-miner is an Android library that performs Proof of Work (PoW) to find the nonce required for posting a TaggedDataBlock on the IOTA network. It allows developers to easily integrate PoW functionality into their Android apps and publish data to IOTAâ€™s Tangle network.

## Add the Dependency
To include this library in your Android project, add the following line to your `build.gradle` file:

```gradle
dependencies {
  ...
  implementation("com.github.AlexandruNicolescu00:android-iota-miner:0.1.1")
  ...
}
```
